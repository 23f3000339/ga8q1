Valuation Methodology

We utilized the Net Present Value (NPV) model to assess the new tech initiative:

\[ NPV = \sum_{t=1}^{T} \frac{R_t}{(1+i)^t} - C_0 \]

Where $R_t$ is net cash inflow, $i$ is the discount rate, and $C_0$ is initial investment.

The math here demonstrates how we calculated the value of our cloud migration. We used a discount rate of 5%.

Automated Risk Analysis

We implemented Python scripts to detect volatility in real-time.


import pandas as pd
import numpy as np

def calculate_volatility(price_data):
    # Calculate daily logarithmic returns
    log_returns = np.log(price_data / price_data.shift(1))
    
    # Annualized volatility
    volatility = log_returns.std() * 252**.5
    return volatility
This is the actual code running on our servers. Lines 4-6 show the logarithmic return calculation. Lines 8-9 convert it to an annualized metric.

Discounted Cash Flow (DCF)

We value projected cash flows using the Discounted Cash Flow model:

\[ V_0 = \sum_{t=1}^{T} \frac{FCF_t}{(1 + r)^t} \]

Where: \[ FCF_t = \text{Free Cash Flow at time } t,\quad r = \text{Discount rate},\quad T = \text{Projection horizon} \]

Here, V0 is the present value of all future free cash flows. The discount rate r incorporates both the risk-free rate and our risk premium. This model underlies our valuation of Q3 and projected Q4 performance.

Risk Calculation Script (Python)

We use the following Python function to compute Value at Risk (VaR):


import numpy as np

def calculate_var(returns, confidence_level=0.95):
    """
    Simple historical VaR using percentile.
    returns: np.array of portfolio returns
    """
    # Mean and standard deviation
    mu = np.mean(returns)
    sigma = np.std(returns)

    # Quantile for given confidence level
    cutoff = np.percentile(returns, (1 - confidence_level) * 100)

    # VaR is the loss at the cutoff
    var_value = mu - cutoff
    return var_value
This example uses a historical VaR approach. The returns array is our distribution of daily portfolio returns. We select a confidence level, compute the percentile cutoff, and interpret the VaR as the expected worst loss at that confidence.

Option-Based Risk Hedging

For currency risk, we reference the Black–Scholes call option formula:

\[ C = S_0 N(d_1) - K e^{-rT} N(d_2) \] with \[ d_1 = \frac{\ln\left(\frac{S_0}{K}\right) + \left(r + \frac{\sigma^2}{2}\right)T} {\sigma \sqrt{T}}, \quad d_2 = d_1 - \sigma \sqrt{T} \]
We don’t derive this in the meeting, but it’s important to mention that our FX hedging uses standard option pricing theory. The parameters are calibrated to market data, and the resulting pricing feeds into our risk dashboard.

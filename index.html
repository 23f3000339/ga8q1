Option Pricing Model

We used the Black-Scholes formula to hedge our currency risks:

\[ C = S_t N(d_1) - K e^{-rt} N(d_2) \]

Where $d_1 = \frac{\ln(S_t/K) + (r + \frac{\sigma^2}{2})t}{\sigma\sqrt{t}}$

This is the standard Black-Scholes model. C is the call option price. S is the current stock price. We used this to calculate risk exposure.

Risk Calculation Script

Python implementation for automated variance tracking:


import numpy as np
import pandas as pd

def calculate_var(portfolio, confidence_level=0.95):
    # Calculate Value at Risk
    mu = np.mean(portfolio)
    sigma = np.std(portfolio)
    
    # Return calculated VaR
    return mu - sigma * np.abs(np.percentile(portfolio, 1-confidence_level))
Here is the Python code we use on the server. Lines 4-5 calculate the mean and standard deviation. Line 8 returns the specific Value at Risk metric.
